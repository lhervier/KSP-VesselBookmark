Proposition de refonte — Vessel Bookmarks

  • Style : repris du cheatsheet PS4 (uGUI / PopupDialog), accent vert #8dbe45.
  • Filtres dans « ⋯ » : recherche, corps, type et case « commentaire » regroupés dans le menu du title bar. Le point vert sur « ⋯ » signale un filtre actif.
  • Deux sections : signets par module de commande / par vaisseau, avec une aide grisée sous chaque titre.
  • Actions hybrides : ▲ ▼ ✕ au survol/sélection de la ligne ; Éditer / Aller / Cibler dans la barre du bas, actives sur la sélection.
  • Vaisseau actif : liseré + fond vert. Cible : titre vert + pastille. Vaisseau disparu : gris italique + pastille rouge.
  • Édition / suppression : calque interne par-dessus le corps (pas de nouvelle fenêtre).
  • Clique une ligne pour la sélectionner, survole-la pour révéler ▲▼✕, ouvre « ⋯ » et « ✎ ».
